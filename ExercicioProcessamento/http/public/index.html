Training Nodejs

<h2>OS command</h2>

Os comandos OS servem para processamento de dados da CPU
são metodos ligados ao sitema operacional.

Alguns metodos usados de Os

  • Os.freemem
  • Os.totalmem
<h2>FS command</h2>

O modulo Fs servem para interação de sistema de arquivos
de uma forma modelada.

Alguns metodos usados de Fs

  • Fs.appendFile
  • Fs.readFile
<h2>Path command</h2>

O modulo Path fornecem utilitarios para trabalhar com caminhos
de arquivos e diretorios.

Alguns metodos usados de Path

  • Path.join
<h2>Events command</h2>

O modulo Events servem como emissores e ouvintes de eventos
que acontecem com algumas API.

Alguns metodos usados de Events

  • events.on
  • events.emit
<h2>HTTP command</h2>

As interfaces HTTP no Node.js são projetadas para oferecer suporte a muitos recursos do protocolo que são tradicionalmente difíceis de usar.
Em particular, mensagens grandes, possivelmente codificadas por partes. A interface tem o cuidado de nunca
armazenar em buffer solicitações ou respostas inteiras, para que o usuário possa transmitir dados.

Alguns metodos usados de HTTP

  • http.createServer
